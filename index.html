GLMplayer

reproductor mp3 codificado en python para sistemas operativos GNU/LINUX

Download this project as a .zip file Download this project as a tar.gz file

Glmplayer es un reproductor mp3 ligero y éficaz. Con un entorno bastante atractivo, es fácil de instalar. Prueba descargando la actual versión en desarrollo. Posteriormente publicaremos paquetes para tus distribuciones. No se ha lanzado ninguna versión estable.

Herramientas adicionales / Aditional tools

Antes de instalar el programa primero debes instalar manualmente los siguiente paquetes - Before installing the software, you must install the following packages:

  • Tweepy ( web del autor )
  • Mutagen ( web del autor )
