GLMplayer

reproductor mp3 codificado en python para sistemas operativos GNU/LINUX

Download this project as a .zip file Download this project as a tar.gz file

GLMPlayer es un reproductor MP3, pensado para permitir conexiones con twitter, y reproducción de video. Puede compartir con tus seguidores la música que estas escuchando en todo momento. La aplicación esta disponible actualmente en fases de desarrollo para sistemas GNU/LINUX que soporten paquetes .deb. Colabora con este proyecto instalando la ultima versión de la aplicación y enviando los reportes de errores al correo william.parras.mendez@gmail.com, gracias por tu apoyo.

GLMPlayer is an MP3 player, designed to allow connections with twitter and video playback. You can share with your followers what you're listening to. The application is currently available in development stages for GNU / Linux systems that support .deb packages. Support this project by installing the latest version of the application and sending Bug's reports to email address william.parras.mendez@gmail.com, thanks for your support.
